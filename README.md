oxide for [Zed](https://zed.dev/), a high-performance, multiplayer code editor.

## Installation

### Development Install

1. Clone this repository.
2. Open Zed's command palette with `cmd-shift-p`.
3. Run `zed: install dev extension`.
4. Select this repository directory.
5. Select `Oxide` from the theme picker.

## Configuration

Set the theme in Zed settings:

```json
{
  "theme": "Oxide"
}
```
